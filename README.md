## ✨ 핵심 기능

### 🔐 계정 기반 데이터 관리

* Google 로그인으로 사용자별 데이터 분리 저장

### 🔄 PC · 모바일 실시간 동기화

* 어디서 입력해도 동일한 데이터 유지
* 브라우저 자동 저장 + Firebase 백업

### 📅 정산 기간 적용

- 일반 달력 기준이 아닌 **[안내] 46시간제 운영 변경에 따른 시프티 변경사항 안내** 공지 기준 적용
- 월~일 단위로 주차 자동 구성

### ⏱️ 근무시간 자동 계산

* 점심시간(12:00 ~ 13:00) **자동 차감**
* 입력만 하면 근무시간 자동 계산

### ➕ 추가 휴게시간 입력

* 저녁, 개인 휴식 시간 등 별도 입력 가능
* `1`, `30`, `130` 입력 시 자동으로 `01:00`, `00:30`, `01:30` 변환

### 📊 시각화 대시보드

* 주차별 근무시간 / 목표 달성률
* 초과 / 부족 여부 한눈에 확인

### 📆 연간 근무 현황

* 1~12월 전체 근무 기록 요약
* 월별 누적 시간 및 목표 비교

## 🧑‍💻 사용 방법

### ⏰ 시간 입력

* 출근 / 퇴근 / 휴게시간 칸 클릭 후 숫자 입력 → Enter
* 입력 예시:

| 입력값    | 변환 결과   |
| ------ | ------- |
| `700`  | `07:00` |
| `30`   | `00:30` |
| `1`    | `01:00` |
| `0730` | `07:30` |

* 점심시간(12:00~13:00)은 자동 차감됩니다
